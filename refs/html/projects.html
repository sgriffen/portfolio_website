Freshman year: CprE 186 Semester project - DND Java

My group and I made a map and character displayer for Dungeons ‘n Dragons. We acomplished this using JFrames and buffered images in Java and created a "fog of war" mechanic that, if turned on, covered the map and as the character model on the screen moved around, by clicking and holding the mouse, the map would uncover itself.
What I learned:

  • How java handles images and layering images on each other
  • Scaling projects to the time-line available
  • Partitioning large projects into smaller, easier projects that are easier to focus on and work on
